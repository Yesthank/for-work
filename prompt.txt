[역할 지정]
너는 엑셀 VBA 및 윈도우 API 전문가야. 보안이 엄격한 사내 PC 환경에서 사용할 '데이터 순차 복사 매크로'를 작성해줘.

[요구 사항]
1. 기능: 엑셀의 특정 범위 데이터를 메모리에 담아두고, 단축키를 누를 때마다 하나씩 순서대로 시스템 클립보드에 복사한다.
2. 데이터 범위:
   - 현재 시트의 B열부터 G열까지 읽는다.
   - 1행부터 데이터가 끝나는 행까지 순회한다.
3. 데이터 가공 (반올림 규칙):
   - 값 자체를 Format 함수를 사용해 텍스트로 고정한다.
   - E열 (5번째 열): 소수점 6째 자리에서 반올림하여 '소수점 5자리'까지 남김 ("0.#####")
   - 나머지 열 (B, C, D, F, G): 소수점 4째 자리에서 반올림하여 '소수점 3자리'까지 남김 ("0.###")
   - 숫자가 아닌 텍스트는 그대로 둔다.
4. 조작 키 (GetAsyncKeyState 사용):
   - [왼쪽 Shift 키 (&HA0)]: 다음 데이터 복사
   - [오른쪽 Shift 키 (&HA1)]: 이전 데이터로 되돌리기 (Undo)
   - [ESC 키]: 매크로 종료
5. 기술적 제약 사항 (매우 중요):
   - 사내 보안 때문에 'MSForms.DataObject'나 'htmlfile' 객체 사용 시 클립보드 오류가 발생함.
   - 반드시 'Windows API (User32, Kernel32)'의 OpenClipboard, SetClipboardData, GlobalAlloc 등을 직접 호출하는 방식으로 코드를 작성할 것.
   - 64비트/32비트 엑셀 호환을 위해 'PtrSafe' 선언을 포함할 것.
   - CPU 점유율 방지를 위해 루프 내에 Sleep과 DoEvents를 넣을 것.

위 요구사항을 모두 만족하는 전체 VBA 코드를 작성해줘.
