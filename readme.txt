
## 📋 [배포용] 엑셀 데이터 순차 복사기 사용 설명서

### 1. 프로그램 소개

이 도구는 엑셀에 정리된 수많은 데이터를 **단축키 하나로 순서대로 복사**해주는 자동화 매크로입니다.
입력할 데이터가 많을 때, `Ctrl+C` → `Alt+Tab` → `Ctrl+V`를 무한 반복하는 번거로움을 해결해 줍니다.

* **별도 설치 불필요:** 엑셀만 있으면 바로 작동합니다. (사내 보안 OK)
* **스마트 반올림:** 지저분한 소수점 데이터를 자동으로 정리해서 복사합니다.

### 2. 주요 기능

* **[왼쪽 Shift]**: 다음 데이터 복사 (누르고 바로 붙여넣기 하세요)
* **[오른쪽 Shift]**: 이전 데이터로 되돌리기 (실수했을 때 사용)
* **[ESC]**: 프로그램 종료
* **자동 데이터 정리:**
* **E열:** 소수점 5째 자리까지 남김 (6째 자리 반올림)
* **나머지(B~G열):** 소수점 3째 자리까지 남김 (4째 자리 반올림)
* **숨겨진 숫자 제거:** 눈에 보이는 숫자 그대로 깔끔하게 복사됨



---

### 3. 최초 설정 방법 (1회만 진행)

이 프로그램은 엑셀 내부 기능(VBA)을 사용하므로, 코드를 한 번 등록해주어야 합니다.

1. 데이터가 들어있는 엑셀 파일을 엽니다.
2. 키보드의 **`Alt` + `F11**` 키를 동시에 누릅니다. (매크로 편집창이 열립니다)
3. 상단 메뉴에서 **[삽입(I)]** → **[모듈(M)]**을 클릭합니다.
4. 나타나는 하얀 빈 창에 **별도로 전달받은 코드(VBA)**를 통째로 붙여넣습니다.
5. 창을 닫고 엑셀 화면으로 돌아옵니다.
6. (선택 사항) 파일을 저장할 때 **'Excel 매크로 사용 통합 문서 (*.xlsm)'** 형식으로 저장하면 설정을 유지할 수 있습니다.

### 4. 사용 방법

1. 엑셀에서 **`Alt` + `F8**` 키를 누릅니다.
2. **`StartSequentialCopy`**를 선택하고 **[실행]**을 클릭합니다.
3. *"준비 끝"* 메시지 창이 뜨면 **확인**을 누릅니다.
4. 이제 데이터를 입력할 웹사이트나 프로그램을 켭니다.
5. **[왼쪽 Shift]**를 툭 누릅니다. (복사 완료)
6. 입력칸에 **`Ctrl` + `V`**로 붙여넣습니다.
7. 다시 **[왼쪽 Shift]** → **`Ctrl` + `V`**를 반복하며 데이터를 입력합니다.
8. 작업이 끝나면 **[ESC]**를 눌러 종료합니다.

### 5. 자주 묻는 질문 (FAQ)

**Q. 매크로 보안 경고가 떠서 실행이 안 돼요.**
A. 엑셀 파일 상단에 노란색 "콘텐츠 사용" 버튼이 있다면 클릭해주세요. 혹은 파일을 `다른 이름으로 저장` → `Excel 매크로 사용 통합 문서(.xlsm)`로 저장 후 다시 열어보세요.

**Q. Shift를 눌렀는데 복사가 안 된 것 같아요.**
A. 엑셀 하단의 초록색 상태 표시줄을 보세요. `복사됨 (1/100) ▶ 0.123` 같은 메시지가 보이면 정상 작동 중입니다.

**Q. 입력 중에 타자를 쳤는데 데이터가 넘어가요.**
A. **[왼쪽 Shift]** 키가 복사 단축키로 설정되어 있습니다. 프로그램 실행 중에는 쌍자음(ㄲ, ㄸ 등)이나 특수문자 입력을 피해주세요. 입력이 필요하면 **[ESC]**로 끄고 입력 후 다시 켜주세요.
