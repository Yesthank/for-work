# 📋 Excel Sequential Copy Macro (엑셀 순차 복사 매크로)

이 프로젝트는 엑셀의 대량 데이터를 **키보드 단축키(Shift)를 통해 순서대로 클립보드에 복사**해주는 VBA 매크로입니다.

반복적인 데이터 입력(Data Entry) 작업 시 `Ctrl+C` → `Alt+Tab` → `Ctrl+V` 과정을 획기적으로 줄여주며, **사내 보안 환경(VBA 클립보드 객체 차단)**에서도 작동하도록 Windows API를 직접 호출하는 방식을 사용합니다.

## ✨ 주요 기능 (Features)

* **⚡ 초고속 순차 복사:** `왼쪽 Shift` 키를 누를 때마다 다음 행의 데이터를 자동으로 클립보드에 저장합니다.
* **↩️ 되돌리기 (Undo):** 실수로 데이터를 넘겼을 때 `오른쪽 Shift` 키로 이전 데이터로 돌아갑니다.
* **🛡️ 강력한 호환성:** `MSForms.DataObject`를 사용하지 않고 `User32.dll` API를 사용하여, 클립보드 오류가 잦은 보안 PC에서도 안정적으로 작동합니다.
* **🎯 스마트 데이터 정제:**
    * 보이는 데이터 값이 아닌, **실제 값**을 가져와 포맷팅합니다.
    * **E열:** 소수점 6째 자리 반올림 → **5째 자리까지 유지** (`0.#####`)
    * **B~G열 (나머지):** 소수점 4째 자리 반올림 → **3째 자리까지 유지** (`0.###`)
    * 숨겨진 소수점(Floating Point Error)을 제거하여 깔끔한 텍스트로 변환합니다.

## 🚀 사용 방법 (Usage)

1.  **매크로 실행:** 엑셀에서 `Alt + F8`을 눌러 `StartSequentialCopy`를 실행합니다.
2.  **데이터 입력:**
    * 입력할 프로그램(웹 브라우저, ERP 등)을 활성화합니다.
    * **`[왼쪽 Shift]`** 키를 '툭' 누릅니다. (복사 완료)
    * **`Ctrl + V`**로 붙여넣습니다.
    * 위 과정을 반복합니다.
3.  **종료:** 작업이 끝나면 **`[ESC]`** 키를 눌러 매크로를 종료합니다.

| 단축키 | 동작 | 비고 |
| :--- | :--- | :--- |
| **Left Shift** | 다음 데이터 복사 | 타이핑 중에는 누르지 않도록 주의 |
| **Right Shift** | 이전 데이터 복사 | 뒤로 가기 (Undo) |
| **ESC** | 프로그램 종료 | 즉시 중단 |

## 🛠️ 설치 방법 (Installation)

이 도구는 별도의 설치 파일 없이, 엑셀 VBA 모듈에 코드를 붙여넣어 사용합니다.

1.  데이터가 있는 엑셀 파일을 엽니다.
2.  `Alt` + `F11`을 눌러 **VBA 편집기**를 엽니다.
3.  메뉴에서 `삽입(Insert)` > `모듈(Module)`을 클릭합니다.
4.  제공된 `SequentialCopy.vba` 코드를 붙여넣습니다.
5.  파일을 **Excel 매크로 사용 통합 문서 (*.xlsm)** 형식으로 저장합니다.

## ⚙️ 설정 수정 (Configuration)

코드 상단의 상수 및 포맷팅 부분을 수정하여 자신의 환경에 맞게 변경할 수 있습니다.

```vba
' === 데이터 시작 위치 설정 ===
Const START_ROW As Long = 1  ' 데이터가 시작되는 행 번호

' === 반올림 규칙 수정 (Format 함수) ===
If c = 5 Then
    ' E열: 소수점 5자리까지 (#을 추가하거나 제거하여 조절)
    currentVal = Format(rawVal, "0.#####")
Else
    ' 나머지 열: 소수점 3자리까지
    currentVal = Format(rawVal, "0.###")
End If
